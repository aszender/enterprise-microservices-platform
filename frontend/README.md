## Running the Files

### JavaScript (ES6)
```bash
node es6.js
node dom-basics.js
```

### TypeScript
```bash
# Install TypeScript globally (if needed)
npm install -g typescript

# Compile TypeScript
tsc typescript-basics.ts

# Run compiled JavaScript
node typescript-basics.js
```

## Interview Tips
- Focus on **closures**, **this keyword**, **prototypes**
- Understand **event delegation** and **event bubbling**
- Know the difference between **==** and **===**
- Understand **hoisting** and **scope** (var vs let vs const)
- Practice **array methods** (map, filter, reduce, forEach)
